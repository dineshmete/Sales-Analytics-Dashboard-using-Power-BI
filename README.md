# Sales Analytics Dashboard using Power BI

## Project Overview

This project presents an interactive Sales Analytics Dashboard developed using Power BI to analyze business sales performance and generate actionable insights for decision-making.

The dashboard provides insights into:
- Total Sales
- Total Cost
- Total Profit
- Total Quantity Sold
- Transactions
- Brand-wise Sales
- Category-wise Quantity
- State-wise Sales Distribution
- Supervisor-wise Performance Analysis

The project helps businesses monitor sales trends, product performance, and operational efficiency using interactive visualizations.

# Project Workflow

## 1. Business Requirement Gathering
- Identified business reporting requirements
- Defined sales KPIs and reporting objectives

## 2. Data Understanding
- Analyzed raw sales datasets
- Studied product, brand, and transaction data

## 3. Data Connection
- Connected datasets using Power BI

## 4. Data Cleaning & Quality Check
Performed:
- Null value handling
- Duplicate removal
- Data validation
- Format standardization

## 5. Data Transformation
- Transformed raw sales data using Power Query
- Created required business attributes

## 6. Data Modeling
- Built table relationships
- Applied star schema concepts
- Optimized data model for reporting

## 7. DAX Calculations
Created:
- Sales Measures
- Cost Metrics
- Quantity Calculations
- KPI Metrics
- Performance Indicators

## 8. Dashboard Development
Designed:
- KPI Cards
- Bar Charts
- Pie Charts
- Geographic Maps
- Interactive Filters
- Business Reports

## 9. Insight Generation
Generated insights related to:
- Top-selling categories
- Brand contribution
- Regional sales distribution
- Business transaction trends
- Sales performance monitoring
